# Shree Sai Krupa Hair Dresses

A premium hair styling and grooming studio located in Khopoli. Established in 2000, offering world-class grooming tailored to your unique style.

## Technologies Used
- HTML / Tailwind CSS
- Node.js / Express
- Three.js (3D Hero Animation)

## Setup
1. Run `npm install` to install dependencies.
2. Start the server with `npm start` or `node server.js`.
3. Open `http://localhost:8081` in your browser.

## Admin Portal
Accessible via `/admin.html`. Provides an interactive dashboard to manage gallery images, view contact messages, and monitor website traffic analytics.
